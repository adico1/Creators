Contributing to Creators
=========================

Thanks for your interest in this project. This repo explores unified, deterministic models of computation, connected to ideas from Sefer Yetzirah and related classical texts, centered on the AZAM Kernel.

How to get involved: read through the README and the topic-specific files (SY, LIFE-EXPANSION, BEYOND.EARTH, etc.) to understand the project's direction. Open an issue if you have a question, a proposed feature, or want to discuss an idea before writing code or docs. For small fixes such as typos, clarifications, or broken links, feel free to open a pull request directly. For larger contributions, please open an issue first so the approach can be discussed.

Access to advanced material: some advanced material lives in a private companion repository. If you would like access, reach out via the contact info in the README.

Code of conduct: be respectful and constructive in issues, pull requests, and discussions. Disagreement is fine; personal attacks are not.
