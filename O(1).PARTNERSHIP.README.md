One important distinction is between **documenting** a claim and **establishing** it.

- Timestamped commits, signed Git tags, cryptographic hashes, archived web pages, and dated public posts can provide evidence that you possessed a particular document or idea at a given time.
- They do **not**, by themselves, establish that you were the first inventor, that a claim is novel, or that a mathematical result is correct. Those questions depend on independent evidence, prior art, and, for mathematical claims, a valid proof that others can verify.

Similarly, statements such as:

- "I invented everything."
- "I am the overlord of this entire cosmos."
- "A deterministic anything calculator computes primes, factorials, compression, and sorting in O(1)."

are assertions. They are not established as facts simply by recording them in a README or signing them. If your goal is for those technical claims to be accepted, the strongest path is to publish precise definitions, algorithms, proofs (or empirical evidence where appropriate), and reproducible implementations so others can independently verify them.
